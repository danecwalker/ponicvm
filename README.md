## üèÅ Getting Started <a name = "getting_started"></a>

### Prerequisites

The project requires you to have Go installed.

```
brew install go
```

### Installing

Clone the repo

```
git clone https://github.com/danecwalker/ponic
```

Run the compiler (currently uses the ./examples/example.pc)

```
go run .
```
